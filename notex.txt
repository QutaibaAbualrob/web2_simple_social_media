Backend part:

    Init app using webapi temp, returns JSON:

        dotnet new webapi -n social_api

    Run app:
        dotnet run

    Install packages:
        dotnet add package Newtonsoft.json

        using Newtonsoft.Json;

        Download 9.0 DOTNET depnednices:
            dotnet add package Microsoft.EntityFrameworkCore --version 9.0.0
            dotnet add package Microsoft.EntityFrameworkCore.SqlServer --version 9.0.0
            dotnet add package Microsoft.EntityFrameworkCore.Tools --version 9.0.0
            dotnet add package Microsoft.AspNetCore.Authentication.JwtBearer --version 9.0.0
            dotnet add package Microsoft.EntityFrameworkCore.Sqlite --version 9.0.0
            dotnet tool install -g dotnet-ef

        Migrations :
            dotnet ef migrations add initialcreate

            dotnet ef migrations add initfix

            dotnet ef database update

        Controllers:

            Test:
                Simple get, post methods to test the database

            Auth:


                Update User Model:
                    Add PasswordHash field to User.cs
                    
                Create DTOs folder:
                    RegisterRequest.cs (Username, Email, Password, Name)
                    LoginRequest.cs (Email, Password)
                    AuthResponse.cs (Token, UserId, Username, Name)

                Configure appsettings.json:
                    Add JWT section with Key, Issuer, Audience

                Update Program.cs:
                    Add JWT authentication middleware
                    Configure TokenValidationParameters
                    Add app.UseAuthentication() before UseAuthorization()

                Create :
                    Register endpoint: Creates user, generates JWT token
                    Login endpoint: Validates credentials, returns JWT token
                    GenerateJwtToken method: Creates signed JWT with user claims

            

            

Running the app:

    back:
        dotnet run

    front:
        npm install
        npm run dev

Frontend part:

    Init app using vite:

        npm create vite@latest social-front
